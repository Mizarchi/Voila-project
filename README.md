# Voila

VOILÀ BEAUTY ROOM

Sistema de Gestión de Spa:

Diseñada para optimizar las operaciones en un spa, gestionando citas, inventario y
almacenamiento. Cuenta con un completo dashboard que muestra estadísticas de clientes a lo
largo del año a través de gráficos interactivos. La aplicación incluye un menú para gestionar citas,
pagos, inventario, productos y almacenamiento, todo integrado en un formato de tabla de datos
fácil de usar.

Frontend: Angular y Angular Material, lo que ofrece una interfaz moderna y responsive.
Backend: JavaScript y TypeScript, garantizando una funcionalidad sólida y un rendimiento
fluido.
